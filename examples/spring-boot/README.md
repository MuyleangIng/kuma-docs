# Spring Boot Example

Public source recipe for a Spring Boot KHQR backend.

Key files:

- `pom.xml`
- `src/main/java/com/example/koma/config/KomaProperties.java`
- `src/main/java/com/example/koma/service/KomaService.java`
- `src/main/java/com/example/koma/web/KomaController.java`

Use `.env.example` as the env contract reference.
