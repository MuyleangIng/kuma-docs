# Vue + Vite Example

This is a standalone Vue sandbox-style example for `koma-khqr`.

Run:

```bash
npm install
cp .env.example .env.local
npm run dev
```

Open `http://localhost:5173`.

Routes included:

- `/`
- `/api/koma-qr`
- `/api/koma-status`
- `/payment/success`
- `/payment/cancelled`
